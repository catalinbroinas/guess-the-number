# Guess the Number - React Game

A simple number guessing game built with React as a portfolio project.  
Initially planned as *Guess the Number / Word*, the project was simplified to focus exclusively on the number-guessing gameplay.

## About

This game offers two game modes:
- Single player
- Multiplayer (two players)

### Single player mode settings:
- Number range
- Player name (optional)
- Number of attempts (optional)

### Multiplayer mode settings:
- Number range
- Player names (optional)

**Note:** In multiplayer mode, the game ends as soon as one player guesses the correct number.

## Technologies

- HTML
- SCSS
- JavaScript
- React
- Vite
- ESLint
